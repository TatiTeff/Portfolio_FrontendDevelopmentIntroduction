Experiencia Laboral y Académica

Actualmente, soy profesora de inglés, pero mi interés por la programación surgió por la busca de crear una web para mi mini emprendimiento que comencé el año pasado. Inicié mi aprendizaje realizando cursos online de forma autodidacta, y se me presentó la oportunidad de realizar este curso de Desarrollo Fronted con WorkerTech PlugIT. Al mismo tiempo, me embarqué en un curso de Python, un acontecimiento que me inspiró a comenzar la Tecnicatura de Programación en ese mismo lugar.

En este momento, estoy experimentando un cambio significativo en mi vida, tanto en el ámbito profesional como en el personal, explorando un nuevo horizonte con el cual me siento muy cómoda. Me emociona la oportunidad de adquirir nuevos conocimientos y siento entusiasmo por las perspectivas que se abren en mi camino en el campo de la programación y el desarrollo web.

Proyectos

Initial Code Lines

El repositorio muestra ejemplos de las primeras líneas de código en HTML y CSS, proporcionando conocimientos esenciales como la estructura de una página HTML, etiquetas para encabezados y párrafos, estilos de texto y selección de fuentes.

Flexbox Image Gallery

En este repositorio, encontrarás un proyecto práctico en el que exploramos las capacidades de Flexbox, incluida la organización de elementos, así como las propiedades y estilos de CSS para crear una galería de imágenes responsiva

Flexbox Positioning Practice

Este repositorio alberga una práctica centrada en el posicionamiento con CSS Flexbox para diseños de web adaptables. Contiene la estructura para crear de tarjetas de contenido con un diseño atractivo y transiciones suaves en los botones.

One Page Web Practice

Este proyecto se enfoca en la creación de una One Page con HTML y CSS, explorando técnicas de gradientes, enlaces suaves y organización de contenido en secciones. Se aplican estilos, márgenes y fuentes personalizadas, incluyendo un diseño con wave.

Festive Flexbox Landing Page

Este repositorio contiene una práctica de una Landing Page con un diseño festivo para la temporada navideña. Se han integrado Google Fonts y se han utilizado atributos Flexbox, Flex Wrap y Gap para crear un diseño flexible y atractivo.

Responsive Web Design

En esta práctica, exploramos el proceso de diseño web responsive, comenzando con un enfoque de 'Desktop First' y luego ajustándolo para adaptarlo de manera efectiva a dispositivos móviles mediante media queries y otras propiedades CSS.
